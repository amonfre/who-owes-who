Instructions for running our code.

If you don't have virtualenv installed on your VM,
pip install virtualenv 

Then, from you terminal window:

1. Go to project folder (cd who-owes-who/djang)
2. command: virtualenv env
3. command: source env/bin/activate
4. pip install Django==1.8
5. python manage.py runserver
6. Go to the server specified on a web browser
7. Explore the website!
